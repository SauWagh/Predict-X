# PredicX – Smart File-Based Prediction System (Django + ML)

PredicX is a Machine Learning-powered web application built using Django, HTML, CSS, JavaScript, NumPy, Pandas, Scikit-Learn, Matplotlib, and Seaborn.
It allows users to upload datasets (CSV, XLSX, DOC, JSON, PDF, etc.), select a target column, visualize data, train ML models, and generate predictions — all through a simple and interactive UI.

## Features

### Upload Multiple File Types
#### Supported formats:

* CSV
* Excel (XLSX)
* JSON
* DOC / DOCX
* PDF
* Text files

### Automatic Data Cleaning & Preprocessing

* Missing value handling
* Encoding categorical features
* Scaling numeric data

### Select Your Target Column

* Choose the column you want to predict and let the system build the ML model automatically.

### Machine Learning Models Used

* Linear Regression
* Logistic Regression
* Random Forest
* Decision Tree
* KNN
* SVM
(Auto selection based on dataset type)

### Graphical Visualization

* Get detailed data analysis graphs using:
* Correlation Heatmap
* Scatter Plot
* Step Plot
* Stack Plot
* Pie Chart
* Histogram
* 3D Scatter Plot
* 3D Triangular Surface

### Public File Upload Section
* Users can upload demo/practice datasets for others to learn and test the application.
